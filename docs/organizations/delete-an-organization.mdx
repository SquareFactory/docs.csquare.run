:::warning

By deleting an organization, it will become unavailable to all users and cannot be recovered. All the projects, project's labs, models and orphans datasets are deleted.

:::

## Quick guide
<VideoPlayer src="https://storage.googleapis.com/cdn.csquare.run/docs/webm/delete-organization.webm" />

## Step-by-step
1. Go to the `Organization` page using the header.
2. Select the organization you want to delete using the dropdown.
3. Click on the `Delete` button.
4. Fill the form: enter the organization's ID on the confirmation field.
5. Click on `Delete permanently`.

That's it, great job!
