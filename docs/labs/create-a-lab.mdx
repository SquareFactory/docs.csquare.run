## Step-by-step
1. [Select the project](../projects/select-a-project) you want to work with.
2. Go to the `Labs` page using the sidebar.
3. Click on the `+` button: `Create lab`.
4. Fill the form:
    1. Enter your new lab's name in the `Lab name` field.
    2. Indicate how many nodes you want to use to run this lab.
5. Click on the `Create` Button.
6. You can now see your new lab in the `Labs` table: well done!
