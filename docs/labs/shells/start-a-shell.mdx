## Step-by-step
1. [Select the project](../../projects/select-a-project) you want to work with.
2. Go to the `Labs` page using the sidebar.
3. Select the lab you want to work with by clicking on its ID.
4. [Start your lab](../start-a-lab).
5. Click on the `+` button.
6. Fill the form.
7. That's it, your new shell is now running!
