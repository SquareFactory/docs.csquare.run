:::success Tip

To know if a project is selected, check the project dropdown in the header: its name should be displayed.

:::

:::info

Once a project is selected, you will be redirected to the **Dashboard** page.

:::

## From the hub

### Quick guide

<VideoPlayer src="https://cdn.csquare.run/docs/webm/select-project-from-hub.webm" />

### Step-by-step

1. Go to the `hub` page (`https://csquare.run/`).
2. Click on the project you want to work with in the `Your project` widget.

That's it: good job!

## From anywhere

### Quick guide

<VideoPlayer src="https://cdn.csquare.run/docs/webm/select-project-from-anywhere.webm" />

### Step-by-step

1. Click on the project dropdown in the header.
2. Click on the project you want to work with.

That's it: easy, isn't it?
