## Step-by-step

:::warning

Only administrators of a project can remove collaborators from said project.

:::

1. #### Select the project in which you want to remove a collaborator.
2. #### Click on the `Delete` button of the corresponding collaborator's line.
3. #### Click on the `Confirm` button.
