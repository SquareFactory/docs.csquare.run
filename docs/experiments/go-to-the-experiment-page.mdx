## Step-by-step

1. [Select the project](../projects/select-a-project.mdx) of your experiment.
2. Go to the `Models` page using the sidebar.
3. Select the model of your experiment by clicking on its ID in the `Available models` table.
4. On the previously selected page, select the experiment you want to re-run by clicking its ID in the `Experiments` table.

You're there, well done!
