Datasets can be removed from projects. This way, the dataset is not deleted from the other projects it is shared with.


## Quick guide

<VideoPlayer src="https://cdn.csquare.run/docs/webm/remove-dataset.webm"/>

## Step-by-step
1. [Select the project](../projects/select-a-project.mdx) you want your dataset to be removed from.
2. Go to the `Datasets` page using the sidebar.
3. Click on the `Remove from project` button, at the end of the corresponding line.

The dataset is not in the table anymore. You did it!
