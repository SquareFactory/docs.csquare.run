### Step 1: Create a project

If you don't have any project, start by creating a new one. To do so, from the home page, click
on Create a new project, and follow the steps. Once created, you will be redirected to your new project's dashboard.

<VideoPlayer src="/img/quick-start/create-a-project.webm"/>

You can switch project whenever you want by clicking on the project dropdown on the header.

### Step 2: Add datasets

Go to the `datasets` page using the sidebar menu, and [create a new dataset](create-a-dataset.md).

You can also [import existing datasets from your other project](import-a-dataset.md).

### Step 3: Create a model

Go to the `models` page using the sidebar menu, and [create a new model](create-a-model.md).
