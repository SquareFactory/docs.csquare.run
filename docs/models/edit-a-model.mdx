## Step-by-step

1. [Select the project](../projects/select-a-project.mdx) that contains the model you want to edit.
2. Go to the `Models` page using the sidebar.
3. Click on the ID of the model you want to edit.
4. Click on the `Edit button`.
5. Fill the form. See the [_Create a model_](create-a-model#fill-the-form) documentation for more info about the form's fields.
6. Click on the `Submit button`.

That's all it took, good job!
