## Step-by-step
1. [Select the project](../projects/select-a-project.mdx) that contains the model you want to edit.
2. Go to the `Models` page using the sidebar.
3. Click on the `Edit` button on the corresponding row.
4. Fill the form. See the [*Create a model*](create-a-model#fill-the-form) documentation for more info about the form's fields.
5. Click on the `Submit button`.
6. That's all it took, good job!
